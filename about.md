# 关于这个博客

## 我是谁

个人投资者，专注美股市场，以期权为主要工具。

## 定位

**个人复盘笔记 + 高频日更 + 数据驱动的期权信号**

核心目标：让未来的自己能快速找到过去的判断，同时让访客能在30秒内理解这个博客在做什么。

## 分析框架

### 怎么看大盘
1. 宏观流动性 (Fed政策、利率)
2. 风险偏好 (VIX、AAII情绪)
3. 板块轮动 (哪类资产领涨)

### 怎么选个股
1. 行业龙头 (竞争优势、护城河)
2. 催化剂 (财报、产品发布、政策)
3. 期权隐含信息 (IV、异动)

### 怎么用期权
1. 方向判断 → 买入 Call/Put
2. 收入增强 → 卖出 Covered Call / Cash Secured Put
3. 风险对冲 → Protective Put / Collar
